# Tarea 6 - Clasificación según conjunto de datos

## Descripción

Este repositorio contiene el desarrollo de la **Tarea 6** de la asignatura **Ingeniería de Datos**, cuyo objetivo es aplicar técnicas de **Machine Learning supervisado** para resolver un problema de clasificación utilizando un dataset real.

En este caso se trabajó con el conjunto de datos:

**Predict Students’ Dropout and Academic Success**

Dataset obtenido desde UCI Machine Learning Repository:  
https://archive.ics.uci.edu/dataset/697/predict+students+dropout+and+academic+success
https://archive.ics.uci.edu/dataset/697/predict%2Bstudents%2Bdropout%2Band%2Bacademic%2Bsuccess

---

## Objetivo del proyecto

Desarrollar un modelo capaz de predecir el estado académico final de estudiantes universitarios, clasificándolos en una de las siguientes categorías:

- **Dropout**: estudiante abandona sus estudios.
- **Graduate**: estudiante finaliza sus estudios exitosamente.
- **Enrolled**: estudiante continúa matriculado.

Para ello se realiza:

- Análisis exploratorio de datos (EDA)
- Definición del problema de clasificación
- Diseño del experimento
- Entrenamiento y evaluación de múltiples modelos
- Comparación de métricas y análisis de resultados
